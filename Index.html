DebussyStore

La verdadera experiencia

DebussyStore es una aplicacion de escritorio realizada para el sistema operativo de Windows, con un diseño fresco y nuevo asi como tambien con colores suaves y faciles de recordar todo esto para poder hacer mas facil reconocible la aplicacion y por igual el logo que cuenta con un diseño simple pero llamativo

Proposito

Poder abarcar todo el terreno del sector secundario en El Salvador

Estructura

El programa se basa en una estructura MVC y en lenguaje C#

Publico

Aspiramos a un publico de empresas con distintas sucursales a nivel nacional

Acerca de DebussyStore:

1¿Porque el nombre Debussy?

El nombre debussy surgio gracias a la idea de ponerle de alguien relacionado en el ambito del espectaculo frances ya hubiera sido este hombre o mujer pero que este haya sido muy influyente y la persona por la cual nos decidimos finalmente fue Claude Debussy un compositor francés, el cual se gano el titulo de primer compositor impresionista aunque termino rechazando el termino.

2¿Como surgio la idea?

Surgio un dia en el colegio cuando hablabamos acerca del proyecto expo y a un compañero se le ocurrio la idea de poder proporcionar una herramienta para una sucursal de ropa que proporcionase la utilidad de que los clientes puedan ver como quedaran las prendas que estos compraran mediante el programa y por igual brindar mas funcionalidades a la sucursal.

3Para que esta pensado el programa

Esta creada para que los clientes puedan llevar a cabos combinaciones diferentes de ropa asi por igual entregar a la sucursal funciones como, administracion de empleados. administracion de inventario. administracion de sucursales por si posee mas a lo largo del territorio nacional, poder actualizar productos, administrar ventas, etc.

MySQL

PhpMyAdmin es una herramienta de software gratuita escrita en PHP , destinada a manejar la administración de MySQL a través de la Web. phpMyAdmin admite una amplia gama de operaciones en MySQL y MariaDB.

Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux y macOS. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, etc.

Visual Studio 2017

Microsoft Visual Studio es un entorno de desarrollo integrado para Windows y macOS. Es compatible con múltiples lenguajes de programación, tales como C++, C#, Visual Basic.

CSS3

CSS es un lenguaje de diseño que permite definir y crear la presentación de un escrito en un lenguaje marcado. Es muy usado para establecer el diseño visual de los documentos web e interfaces de usuario en HTML.

HTML 5

HTML 5 es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 específica variantes de sintaxis para HTML: HTML, conocida como HTML5, otra variante XHTML conocida como sintaxis XHTML 5.

C Sharp

C# es un lenguaje de programación multiparadigma desarrollado y estandarizado por la empresa Microsoft como parte de su plataforma .NET, que después fue aprobado como un estándar por la ECMA e ISO.

JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

Bootstrap

Bootstrap es un framework front-end utilizado para desarrollar aplicaciones web y sitios mobile first, o sea, con un layout que se adapta a la pantalla del dispositivo utilizado por el usuario.
